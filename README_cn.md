## 🎯 项目简介

正和健康监测系统是一款创新的智能健康管理平台，融合了先进的深度学习算法和传统医学知识，为用户提供全方位的健康监测服务。系统采用模块化设计，支持三种独立工作模式：

- **姿态监测模式**：实时检测人体关键点，识别摔倒等异常姿态
- **药品识别模式**：OCR识别药品包装文字，匹配用药建议数据库
- **心率监测模式**：基于rPPG技术的非接触式心率测量

## ✨ 核心功能

### 1. 人体姿态检测与摔倒预警

- 基于YOLOv11 Pose模型的高精度人体关键点检测
- 支持多人同时检测，实时计算人体宽高比判断摔倒状态
- BPU硬件加速推理，显著提升处理速度
- 可视化标注关键点、边界框和状态标签

### 2. 药品OCR智能识别

- 集成EasyOCR引擎，支持中英文混合文本识别
- 内置1000+种常见药品数据库（med-txt.txt）
- 智能过滤无关文本，精准匹配药品名称和用药建议
- 异步队列处理，避免阻塞主线程

### 3. 非接触式心率监测

- 基于远程光电容积脉搏波（rPPG）技术
- POS（Plane Orthogonal to Skin）算法抗运动干扰
- 频域+时域双重验证提高准确性
- 信号质量指标（SQI）评估数据可靠性
- Haar级联分类器人脸检测与ROI提取

### 4. Web实时监控界面

- 现代化Glassmorphism UI设计
- 响应式布局，支持多终端访问
- 实时视频流推送（MJPEG格式）
- 动态数据更新和模式切换

## 🏗️ 技术架构

### 核心技术栈

| 模块 | 技术选型 | 说明 |
|------|---------|------|
| **后端框架** | Flask 2.0+ | 轻量级Web服务器 |
| **姿态检测** | YOLOv11 Pose | 高性能人体关键点检测 |
| **OCR引擎** | EasyOCR | 多语言文本识别 |
| **心率算法** | rPPG (POS) | 远程光电容积脉搏波 |
| **人脸检测** | OpenCV Haar Cascade | 快速人脸定位 |
| **硬件加速** | Hobot BPU | 地平线机器人处理器 |
| **前端UI** | HTML5 + CSS3 | 现代Web技术 |
