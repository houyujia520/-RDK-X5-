# 快速启动指南

## 🚀 5分钟快速开始

### 步骤1: 安装依赖

```bash
pip install opencv-python numpy scipy flask flask-cors
```

**如果使用RDK板子**,额外安装:
```bash
pip install hobot-vio
```

### 步骤2: 运行程序

```bash
python main.py
```

你会看到类似输出:
```
[INFO] 初始化RDK摄像头...
[INFO] RDK摄像头已打开: 640x480@30fps
Starting Heartbeat Monitoring Thread...
 * Running on http://0.0.0.0:5000
```

### 步骤3: 获取设备IP地址

**Windows:**
```bash
ipconfig
```

**Linux/Mac:**
```bash
ifconfig
```

找到你的IPv4地址,例如: `192.168.43.216`

### 步骤4: 修改HTML文件中的IP地址

在你的HTML文件中,将所有 `192.168.43.216` 替换为你的实际IP地址:

```javascript
// 修改这三处
http://YOUR_IP:5000/video_feed
http://YOUR_IP:5000/api/status
http://YOUR_IP:5000/api/drug
```

### 步骤5: 在浏览器中打开HTML文件

直接双击HTML文件或在浏览器中打开即可。

## 🔧 故障排除

### 问题1: 提示端口被占用

**解决方案**: 修改web_stream.py中的端口号
```python
app.run(host='0.0.0.0', port=5001, threaded=True)  # 改为5001或其他端口
```

同时修改HTML中的端口号。

### 问题2: 网页显示"设备离线"

**检查清单**:
- [ ] 确认程序正在运行(查看控制台输出)
- [ ] 确认IP地址正确
- [ ] 确认防火墙允许5000端口
- [ ] 尝试访问 `http://YOUR_IP:5000/health` 看是否返回 `{"status": "running"}`

### 问题3: 心率一直显示"--"

**可能原因**:
1. **光线不足**: 增加照明
2. **距离太远**: 保持30-50cm距离
3. **额头遮挡**: 确保额头无刘海、帽子等遮挡
4. **等待时间不够**: 程序启动后需等待10-15秒积累数据

### 问题4: RDK摄像头不可用

这是正常的!如果在普通PC上运行,系统会自动切换到普通摄像头。看到以下提示无需担心:
```
[WARN] hobot_vio库未找到,将使用普通OpenCV摄像头
[INFO] 初始化普通摄像头...
```

## 📊 验证系统工作

### 方法1: 手指测试(推荐)

用手指直接接触摄像头镜头,此时心率应该显示较高数值(因为信号很强)。如果准确,说明算法正常。

### 方法2: API测试

在浏览器中访问:
```
http://YOUR_IP:5000/api/status
```

应该返回JSON数据:
```json
{
  "bpm": 75.5,
  "count": 45,
  "status": "online"
}
```

### 方法3: 视频流测试

在浏览器中访问:
```
http://YOUR_IP:5000/video_feed
```

应该能看到实时视频画面。

## 💡 优化建议

### 获得更准确的心率

1. **光照条件**
   - ✅ 使用自然光或直流LED灯
   - ❌ 避免荧光灯(有50/60Hz频闪)
   - ❌ 避免强光直射面部

2. **拍摄角度**
   - ✅ 正对摄像头
   - ✅ 面部居中
   - ❌ 避免侧脸或仰头

3. **环境要求**
   - ✅ 保持头部静止
   - ✅ 关闭摄像头自动曝光(AE)和自动白平衡(AWB)
   - ❌ 避免剧烈运动后立即测量

4. **测量时长**
   - 静息状态: 测量30秒取平均值
   - 运动后: 静息1-2分钟后再测量

## ❓ 常见问题

**Q: 可以在手机上访问吗?**  
A: 可以!只要手机和电脑在同一WiFi网络,用手机浏览器访问 `http://电脑IP:5000` 即可。

**Q: 支持多人同时检测吗?**  
A: 当前版本只检测第一个识别到的人脸。

**Q: 心率范围是多少?**  
A: 算法设计范围为40-220 BPM,覆盖正常人心率范围。

**Q: 数据会保存吗?**  
A: 不会,所有数据仅在内存中临时处理,程序关闭后数据丢失。
